1. WHAT IS DAY ARCHITECT?
================================================================================

  Day Architect is a mobile app (Flutter) that lets students plan their day,
  block distracting social media during focus sessions, and track sleep —
  all in one place. It is designed specifically for Filipino college students.

  Core philosophy: "PLAN IT · FOCUS ON IT · SLEEP ON IT"

  Key features:
    • Daily schedule with categorized task blocks (Class, Focus, Routine, Org, Sleep)
    • Pomodoro-style Focus Mode with a visual timer ring & app blocking display
    • Wind-down mode with Do Not Disturb, dim social apps, and alarm settings
    • Progress dashboard with streak calendar, focus hours chart, sleep consistency
    • Local SQLite database for offline-first persistence
    • Local notifications for session completions and wind-down reminders
    • Seed data pre-loaded for immediate testing

5. DATA MODELS
================================================================================

  5a. Task (focus_session.dart)
  ─────────────────────────────────────────────────────────────────────────────
    • id              — int? (auto-increment PK)
    • title           — String (name of the task)
    • time            — String (e.g. "8:00")
    • chipLabel       — String (category: Routine, Class, Focus, Org, Sleep)
    • accentColor     — int (Color.value for the card accent border)
    • chipBg          — int (Color.value for the category chip background)
    • meta            — String? (optional notes, e.g. "CpE 205 · Rm 302")
    • done            — bool (completion status)
    • date            — String (ISO-8601 yyyy-MM-dd)
    • sortOrder       — int (ordering within a day)

  5b. FocusSession (focus_session.dart)
  ─────────────────────────────────────────────────────────────────────────────
    • id              — int? (auto-increment PK)
    • title           — String (session name)
    • taskId          — int? (FK to tasks.id)
    • durationMinutes — int (minutes focused)
    • interruptions   — int (number of interruptions, default 0)
    • date            — String (ISO-8601 yyyy-MM-dd)

  5c. SleepLog (sleep_log.dart)
  ─────────────────────────────────────────────────────────────────────────────
    • id                  — int? (auto-increment PK)
    • date                — String (ISO-8601 yyyy-MM-dd)
    • sleepDurationMinutes — int (minutes slept)
    • goalDurationMinutes  — int (target minutes, default 450 = 7h 30m)
    • formattedDuration   — getter (e.g. "7h 10m")
    • fractionOfGoal      — getter (0.0–1.0+)

6. SERVICES
================================================================================

  6a. DatabaseHelper (database_helper.dart) — Singleton
  ─────────────────────────────────────────────────────────────────────────────
    • Local SQLite database stored at: day_architect.db
    • 4 tables: tasks, focus_sessions, sleep_logs, preferences
    • Database version: 3 (with migration support)
    • CRUD for all models:
        - Tasks: insert, update, delete, getTasks (by date)
        - Focus Sessions: insert, getFocusSessions, getTotalFocusMinutes,
          getFocusMinutesByDay, getAverageFocusMinutesByDayOfWeek
        - Sleep Logs: insert, getSleepLogs, getSleepLog, getAverageSleepMinutes
        - Preferences: getPreference, setPreference, getAllPreferences
    • Special queries:
        - getStreak() — consecutive days with completed tasks
        - getAverageFocusMinutesByDayOfWeek() — per-day-of-week averages
    • Seed data: 5 tasks, 3 focus sessions, 7 sleep logs pre-populated

  6b. NotificationService (notification_service.dart) — Singleton
  ─────────────────────────────────────────────────────────────────────────────
    • Uses flutter_local_notifications
    • 2 notification channels: focus_sessions, wind_down
    • Methods:
        - init() — initialize channels, request Android 13+ permission
        - showNotification() — fire immediate notification
        - showSessionComplete() — "🎯 Session Complete! You focused for Xh Ym"
        - scheduleWindDown() — schedule a zoned bedtime reminder
        - cancelWindDown() — remove scheduled wind-down notification

      textSecondary= #8A87B0 (muted lavender)
      textMuted    = #5A5880 (dimmed purple)
      textAmberLight = #C9A98E
      textLavender = #C9C4E0

    Surfaces:
      cardSurface = rgba(255,255,255,0.055)
      cardBorder  = rgba(255,255,255,0.055)

    Chip backgrounds (15% opacity):
      chipAmberBg = #33E8935B
      chipCoralBg = #33F2A488
      chipSageBg  = #339CAF94
      chipPlumBg  = #336E6A99

  7b. Gradients (app_theme.dart — AppGradients)
  ─────────────────────────────────────────────────────────────────────────────
    background    = bgLight → bgMid → #1A1B38
    accentButton  = accent → accentSoft

  7c. Typography (app_theme.dart — AppTextStyles)
  ─────────────────────────────────────────────────────────────────────────────
    Heading: Lora, 24px, w700 (variable size)
    Eyebrow: Lora italic, 15px, amber-light color
    Body:    Poppins, 13px, w400 (variable)
    Label:   Poppins, 11px, w700, 1.0 letter-spacing, muted color

  7d. Accessibility
  ─────────────────────────────────────────────────────────────────────────────
    • Text scaling capped between 0.8x and 1.3x to prevent layout breakage
    • ColorScheme ensures sufficient contrast for onPrimary/onSurface
    • Debug show banner disabled
